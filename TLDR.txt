## O que fazer:

1. **Prepare a build** (5 min)
   ```bash
   flutter clean && flutter pub get
   flutter build windows --release
   ```

2. **Teste localmente** (2 min)
   - Abra o app com seu banco atual
   - Verifique dados aparecem normalmente
   - Pronto!

3. **Crie ZIP para distribuir** (1 min)
   - Pasta: `build/windows/runner/Release/`
   - Inclua: `GUIA_ATUALIZACAO_CLIENTE.md`

4. **Envie ao cliente** (1 min)
   - Email + arquivo ZIP
   - Pronto!

**Tempo total:** ~ 10 minutos
